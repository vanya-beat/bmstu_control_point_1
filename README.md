# Проект для проверочных работ студентов (C)

## Структура проекта
- `student_lib/` - статическая библиотека на C с заданиями
- `tests/` - автоматические тесты для проверки решений (C++ с Google Test)

## Сборка проекта

```bash
mkdir build
cd build
cmake ..
cmake --build .
```

## Запуск тестов

```bash
cd build
ctest --output-on-failure
```

Или запустите напрямую для просмотра баллов:
```bash
./tests/student_tests
```

## Система оценивания

Каждое задание оценивается автоматически:
- Задания 1-10: по 0.5 балла каждое (всего 5 баллов)
- Задание 11 (скобочная последовательность): 5 баллов
- Задание 12 (нормализация Unix пути): 5 баллов
- Задание 13 (обратная польская нотация): 10 баллов

**Максимально:** 25 баллов

### Критерии оценки:
- 20+ баллов - ОТЛИЧНО (5)
- 15-19.9 баллов - ХОРОШО (4)
- 10-14.9 баллов - УДОВЛЕТВОРИТЕЛЬНО (3)
- < 10 баллов - НЕУДОВЛЕТВОРИТЕЛЬНО (2)

Баллы начисляются пропорционально количеству пройденных тестов в каждом задании.

## Для студентов

Реализуйте функции в файле `student_lib/student_tasks.c`. Все объявления находятся в `student_lib/student_tasks.h`.

**Важно:** Библиотека написана на чистом C (стандарт C11).

## Задания

1. `int add(int a, int b)` - сложение двух чисел (0.5 балла)
2. `int factorial(int n)` - вычисление факториала (0.5 балла)
3. `bool isPrime(int n)` - проверка на простое число (0.5 балла)
4. `int findMax(const int* arr, int size)` - поиск максимума в массиве (0.5 балла)
5. `void reverseString(const char* str, char* reversed)` - переворот строки (0.5 балла)
6. `bool isPalindrome(const char* str)` - проверка на палиндром (0.5 балла)
7. `void bubbleSort(int* arr, int size)` - сортировка пузырьком (0.5 балла)
8. `int gcd(int a, int b)` - НОД двух чисел (0.5 балла)
9. `unsigned int fibonacci(unsigned int n)` - число Фибоначчи (0.5 балла)
10. `void intToHexString(unsigned int number, char* hexString)` - перевод в hex (0.5 балла)
11. `bool isValidParentheses(char symbol, ...)` - проверка скобок (5 баллов)
12. `void normalizeUnixPath(const char* path, char* normalizedPath)` - нормализация пути (5 баллов)
13. `int reversePolishNotation(const char* expression)` - обратная польская нотация (10 баллов)

## Для преподавателей

Система автоматически подсчитывает баллы и выводит детальный отчет после запуска тестов.
Результаты включают:
- Статус каждого задания
- Количество пройденных тестов
- Набранные баллы
- Итоговую оценку

Для изменения баллов отредактируйте файл `tests/score_calculator.h`.
