# Проект для проверочных работ студентов (C/C++) — Вариант 3

## Структура проекта
- `student_lib/` - статическая библиотека с заданиями на C и C++
  - `student_tasks.c/h` - задания на C (массивы, базовые алгоритмы)
  - `student_tasks.cpp/hpp` - задания на C++ (контейнеры STL)
- `tests/` - автоматические тесты для проверки решений (C++ с Google Test)

## Сборка проекта

```bash
mkdir build
cd build
cmake ..
cmake --build .
```

## Запуск тестов

```bash
cd build
ctest --output-on-failure
```

Или запустите напрямую для просмотра баллов:
```bash
./tests/student_tests
```

## Система оценивания

Каждое задание оценивается автоматически. Проект включает два набора заданий:

### БЛОК A: Задания на C (15 заданий на массивы) - 10 баллов
### БЛОК B: Задания на C++ (10 заданий на контейнеры STL) - 17.5 баллов

**Максимально:** 27.5 баллов

### Критерии оценки:
- 23+ баллов - ОТЛИЧНО (5)
- 18-22.9 баллов - ХОРОШО (4)
- 13-17.9 баллов - УДОВЛЕТВОРИТЕЛЬНО (3)
- < 13 баллов - НЕУДОВЛЕТВОРИТЕЛЬНО (2)

Баллы начисляются пропорционально количеству пройденных тестов в каждом задании.

## ⚠️ ВАЖНО для заданий на C++

**Выбор контейнера критичен!** В заданиях на C++ НЕ указывается, какой контейнер использовать.
Вы должны сами выбрать правильный контейнер (vector, map, unordered_map, list, pair и их комбинации).

**Тесты проверяют не только корректность, но и производительность!**
- При неправильном выборе контейнера тесты НЕ ПРОЙДУТ по времени
- Например: использование vector с O(n) вставкой в начало вместо list провалит тесты
- Использование поиска O(n²) вместо hash map провалит тесты на больших данных

### Задания на C:
Реализуйте функции в файле `student_lib/student_tasks.c`. Все объявления находятся в `student_lib/student_tasks.h`.
**Важно:** Используйте чистый C (стандарт C11). Для динамического выделения памяти: `malloc` / `free`.

### Задания на C++:
РеалБЛОК A: Задания на C (работа с массивами)

См. файл `student_lib/student_tasks.h` для полного списка заданий на C.
Всего 15 заданий на массивы, циклы, базовые алгоритмы.

### БЛОК B: Задания на C++ (контейнеры STL) - 17.5 баллов

**Важно:** В этих заданиях НЕ указывается явно, какой контейнер использовать!
Выбор правильного контейнера - часть задания. Неправильный выбор приведет к TLE (превышение времени).

1. **removeDuplicates** (1 балл) - Удаление дубликатов с сохранением порядка
   - Подсказка: нужна быстрая проверка "встречался ли элемент"

2. **twoSum** (1.5 балла) - Найти пару элементов с заданной суммой
   - Подсказка: O(n²) не пройдёт, нужна структура для быстрого поиска

3. **groupAnagrams** (2 балла) - Группировка анаграмм
   - Подсказка: нужен ключ для идентификации анаграмм

4. **topKFrequent** (2 балла) - K наиболее частых элементов
   - Подсказка: сначала частоты, потом выбор топ-k

5. **nextGreaterElement** (1.5 балла) - Следующий больший элемент
   - Подсказка: предобработка + быстрый lookup

6. **LRUCache** (3 балла) - LRU кеш с O(1) операциями
   - Подсказка: нужна комбинация двух контейнеров

7. **mergeKSortedLists** (3 балла) - Слияние K отсортированных списков
   - Подсказка: простая конкатенация + сортировка может не пройти

8. **firstUniqChar** (1 балл) - Первый уникальный символ
   - Подсказка: два прохода - подсчёт и поиск

9. **sortedFrequency** (1.5 балла) - Частоты отсортированные по ключу
   - Подсказка: какой контейнер автоматически сортирует?

10. **MiddleInsertionList** (2 балла) - Эффективная вставка в середину
    - Подсказка: вставка в vector в середину - O(n), нужно лучше

Подробное описание каждого задания см. в `student_lib/student_tasks.hpp`
17. `int* matrixTranspose(const int *matrix, int rows, int cols)` - транспонирование матрицы, результат через malloc (3 балла)
18. `int* spiralOrder(const int *matrix, int rows, int cols, int *resultSize)` - спиральный обход матрицы, результат через malloc (4 балла)
19. `char** stringSplit(const char *str, char delimiter, int *count)` - разбиение строки по разделителю, все строки и массив через malloc (5 баллов)
20. `int* polynomialMultiply(const int *a, int sizeA, const int *b, int sizeB, int *resultSize)` - умножение полиномов, результат через malloc (5 баллов)
